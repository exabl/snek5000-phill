# snek5000-phill

Nek5000 [phill
example](https://github.com/KTH-Nek5000/KTH_Examples/tree/master/phill_STAT)
executed via [snek5000](https://github.com/exabl/snek5000).


## Quick start

```sh
pip install -e '.[tests]'
# Run simple tests: including compilation
pytest
# Run slow tests: launches simulation
pytest --runslow
```
